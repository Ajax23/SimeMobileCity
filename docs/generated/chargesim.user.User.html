    User¶

    class chargesim.user.User(probability={}, ident='')¶

    This class defines a user.

    Parameters:
    probability : dictionary, optional

    Dictionary containing leave probabilities for each day - if empty, all hours have the same probability

    ident : string, optional

    Optional user name
